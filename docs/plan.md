# 大模型自动评价智能体BenchForge

# 参考资料
- BenchAgents：框架参考
- AutoBencher：主题扩展和多语言参考，使用wiki检索相关文档
- YourBench：根据文档生成题目参考
- EvalTree：能力描述参考，生成能力树和弱点树


# 目标
框架：多智能体框架，参考BenchAgents（可进行多轮次评估）
- 规划智能体：
  - 根据用户输入生成总的评估蓝图（总蓝图是不是需要共享），
  - 生成送入其他智能体的层次计划
  - 收到反馈之后，根据反馈生成新计划指导其他智能体的行为
- 题目生成智能体：根据规划智能体给定的要求生成题目
  - 规划智能体给定主题和其他关于题目的要求，题目类型、比率、多语言等
  - 根据主题仿照autobencher进行主题自适应搜索和具体页面检索
  - 获取页面之后，参考yourbench进行题目生成
  - 在生成题目的过程中参考evaltree对题目添加能力描述属性，为后期构建能力树创建基础
- 题目验证智能体
  - 根据规划智能体的要求对题目进行验证，参考yourbench中的题目验证方法
  - 总结现有的题目状态，如果题目数目、难度等不满足要求，向规划智能体反馈，引导题目生成智能体继续生成题目
- 模型评估智能体
  - 根据规划智能体的计划使用生成的基准对模型进行评估
  - 根据autobencher的要求统计信息，反馈给规划智能，进行下一层的主题筛选
  - 参考其他详细记录评估过程：评估时间、token消耗、模型调用次数等
- 分析智能体
  - 根据题目的能力描述和评估结果，参考evaltree生成能力树和弱点树
